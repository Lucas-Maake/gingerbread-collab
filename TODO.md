# Gingerbread House Builder - Future Improvements

A list of potential features and improvements to consider for future development.

## Visual & Aesthetic

- [x] Animated chimney smoke particles
- [x] Warm interior glow from windows/doors
- [x] Frosting drip effects on roof edges
- [x] Day/night lighting toggle
- [ ] Improved shadows and reflections on candy pieces
- [x] Procedural textures for more pieces

- [ ] Copy/paste pieces
- [ ] Duplicate pieces
- [ ] Multi-select (shift+click to select multiple)

- [ ] User roles (host can lock pieces, kick users)

- [ ] Activity log showing who placed/moved what
- [ ] Spectator mode
